# employee-performance-analysis-statistics-project
Employee Performance Analysis project using Python, Statistics, Probability, Distribution Analysis, and Linear Algebra on a dataset of 4000 employee records. Includes data visualization, Gaussian distribution fitting, Q-Q plots, correlation heatmap, probability analysis, and statistical insights using Pandas, NumPy, Matplotlib, SciPy, and Seaborn.
# Employee Performance Analysis Project
## Overview
This project performs Employee Performance Analysis using Statistics, Probability, Distribution Analysis, and Linear Algebra on a dataset containing 4000 employee records.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Jupyter Notebook

## Features

- Mean, Median, Mode
- Variance and Standard Deviation
- Probability and Conditional Probability
- Gaussian Distribution
- Q-Q Plot
- Correlation Heatmap
- Linear Algebra Operations
- Statistical Insights

## Dataset

The dataset contains:
- Employee_ID
- Department
- Age
- Salary
- Projects_Completed
- Working_Hours
- Performance_Score
- Promotion_Status

## Conclusion
The project demonstrates the practical implementation of Mathematics and Advanced Statistics concepts in real-world employee performance analysis.
